## DecisionTree — cache eviction policy

<DecisionTree

  caption="Which cache eviction policy?"
  root={{
    question: 'Is your access pattern dominated by recency (recently used items are reused soon)?',
    branches: [
      {
        label: 'Yes, recency matters',
        next: { leaf: { recommendation: 'LRU', detail: 'Evict the least-recently-used entry. Great default for temporal locality.' } },
      },
      {
        label: 'No — frequency matters more',
        next: {

          branches: [
            {
              label: 'Yes, clear hot keys',
              next: { leaf: { recommendation: 'LFU', detail: 'Evict the least-frequently-used entry to keep hot keys resident.' } },
            },
            {
              label: 'No clear pattern / want simplicity',
              next: { leaf: { recommendation: 'FIFO or Random', detail: 'When neither recency nor frequency dominates, a simple FIFO/Random policy is cheap and predictable.' } },
