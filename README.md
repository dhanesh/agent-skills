# Agent Skills

Personal Agent Skills repository for installation through the open `skills` CLI.

## Install

```bash
npx skills add Dhanesh/agent-skills --skill tmux-agent-herdr-lite
```

## Skills

- `tmux-agent-herdr-lite` — tmux cockpit with Zellij-like human ergonomics and Herdr-like agent supervision.

See each skill directory's `SKILL.md` for usage and prerequisites.
